# API Documentation

## Currently Implemented

### GET `/static/:filename`
Get a static track file by filename.

### GET `/tracks`
List all track info.

---

### Upcoming Features

#### GET `/tracks/:id`
Get track info by id.

#### POST `/tracks`
Create a new track.

#### PUT `/tracks/:id`
Update an existing track.

#### DELETE `/tracks/:id`
Delete a track.
